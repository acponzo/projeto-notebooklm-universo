# 🌌 Mistérios do Universo: Relatividade, Buracos Negros e Teoria das Cordas

## Contexto e Objetivo

Sempre fui fascinada por física e pela forma como grandes pensadores, como Einstein e Stephen Hawking, foram capazes de teorizar conceitos extremamente complexos muito antes de serem comprovados pela ciência, como é o caso dos buracos negros.

Apesar de serem temas incríveis, esses assuntos costumam ser difíceis de entender, principalmente para quem está começando. Por isso, este projeto tem como objetivo transformar conceitos avançados da física moderna em explicações mais simples, acessíveis e organizadas.

Utilizando o NotebookLM como ferramenta de apoio, a proposta é estudar conteúdos confiáveis, aplicar técnicas de engenharia de prompts e construir um material que facilite a compreensão desses temas, permitindo que qualquer pessoa consiga entender, ao menos em nível básico, como funciona o universo.
